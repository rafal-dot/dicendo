# Privacy Policy – dicendo for Android

**Last updated:** 21 February 2026

dicendo is an offline password generator that works entirely on the user’s device.

## 1. No Data Collection

The application:
- does not collect personal data  
- does not transmit any data to external servers  
- does not require user accounts  
- does not contain advertisements  
- does not use tracking or analytics tools  

All operations are performed locally on the user’s device.


## 2. Generated Passwords

By default, generated passwords are not stored or retained by the application.

If the user enables the option to store generated passwords, the data is saved exclusively on the device and protected using Android-recommended security mechanisms (encrypted local storage).


## 3. Permissions

The application uses only the minimum permissions necessary for its operation.  
It does not access contacts, location data, microphone, camera, or other sensitive information.


## 4. Data Sharing

Since the application does not collect or transmit data, no user data is shared with third parties.


## 5. Contact

If you have any questions regarding this Privacy Policy, please contact the developer using the contact details provided on Google Play.
